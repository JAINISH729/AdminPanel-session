## Project Description

This is a Node.js Admin Panel project developed using the MVC (Model-View-Controller) architecture. The project allows administrators to manage events with complete CRUD (Create, Read, Update, Delete) operations. It also supports image uploads using Multer and stores data in MongoDB.

The purpose of this project is to demonstrate backend development skills, database integration, file handling, and MVC architecture implementation.

---

## Technologies Used

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Frontend
- EJS
- HTML
- CSS
- JavaScript

### Other Packages
- Multer (Image Upload)
- Nodemon
- Path
- FS

---

## Features

- Admin Dashboard
- Add New Event
- View Events
- Update Event Details
- Delete Events
- Upload Event Images
- MongoDB Database Integration
- MVC Architecture
- Responsive Admin Interface

---

## How This Project is Made

1. Created the project structure using MVC Architecture.
2. Configured Express.js server.
3. Connected MongoDB database using Mongoose.
4. Created Event Model for database operations.
5. Developed Controllers to handle business logic.
6. Created Routes for managing requests.
7. Designed EJS views for user interface.
8. Implemented Multer for image uploads.
9. Added CRUD functionality for Events.
10. Tested all functionalities and database operations.
